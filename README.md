# Favourite Movies Showcase

A React app showcasing my favourite movies built with Vite

## Links
- **GitHub Repository:** [https://github.com/yahyakalvium/movie_showcase.git]
- **Live Website:** [https://movieshowcasewaseem.netlify.app/]

## Run Locally
1. Clone the repository by using `git clone`
2. Run `npm install`
3. Run `npm run dev`
